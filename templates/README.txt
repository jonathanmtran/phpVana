Don't modify the CoolWater directory for it will be the template that has
all of the features of phpVana.

Also, in the event of your default template not be available, this template will
serve as a fallback for you so that you know that something is wrong

If there is a new feature or fix with phpVana that involves the layout, the
CoolWater template should serve as an example for you or your template
designer.

If you want to use the CoolWater template as your website while adjusting
it to your liking, please make a copy of that folder and name it something to
your taste.

Since this template is avaliable at Styleshout.com, please keep the link inside the layout.
